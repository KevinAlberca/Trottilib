Côté utilisateur

  • Inscription

    L'inscription requiert le nom, le prénom, le lieux de vie de l'utilisateur (adresse, code postal et ville), une date de naissance, et pour s'identifier : une adresse e-mail et un mot de passe.

    Processus :
    • Comptage de nombre d'utilisateur ayant cette adresse e-mail en base de donnée
    • Si aucun utilisateur n'est trouvé, on procède à l'inscription, sinon, on lui indique l'erreur
    • Si aucun utilisateur n'est trouvé : chiffrement du mot de passe
    • Ajout en base de donnée de l'utilisateur
    • On ajoute ces informations dans notre base de donnée pour les recherches (partie back-office)
  • Connexion

    La connexion sert à l'utilisateur de s'abonner et de gérer son abonnement

    Processus :
    • On regarde combien d'utilisateur utilise l'adresse e-mail et le mot de passe renseigné en base de donnée
    • Connexion de l'utilisateur, via une SESSION
  • Mon Espace
    • Récapitulatif des informations
    • Récapitulatif de l'abonnement
    • Changement d'adresse

Côté abonnement

  • Voir les abonnements disponibles
  • Récupère en base de donnée toutes les informations liées aux abonnements
  • Affiche les informations sur la page
  • Voir son abonnement
  • Récupère en base de donnée, toutes les informations concernant les abonnements, correspondant à l'utilisateur

Côté panier

  • Ajouter au panier
  • Ajoute en SESSION l'article souhaité
  • Vider le panier
  • Vide la SESSION (unset)
  • Acheter le panier
  • Redirige vers la sandbox paypal pour procéder au paiement en ligne
